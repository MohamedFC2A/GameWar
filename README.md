# GameWar

Godot 4 offline Android tank arena prototype.

Core loop:
- short stage-based battles against AI
- post-stage upgrade screen
- offline save/load
- mobile-first controls

How to run:
- Open the folder in Godot 4.x.
- Set `scenes/main.tscn` as the main scene if needed.
- Run on desktop first, then export to Android APK from Godot.

Notes:
- This project no longer depends on Unity for gameplay.
- The old Unity scaffold remains in the repo only as legacy material and can be ignored.
